How to use TabChat

After logging in try selecting marks or changing the Genre Quick Filter (remember to hit apply). The chat column will show a summary of your interaction and who performed that interaction.

After the chat column has messages about interactions, try clicking the blue link in one of the messages. The viz will select those same marks or filter to those same genres.

Try having a friend log in and interact with the viz at the same time. If you don't have any friends, open up a second browser window and point it to this same URL. You will be able to 'chat' with Tableau!
